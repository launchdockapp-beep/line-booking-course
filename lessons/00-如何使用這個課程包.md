# 如何使用這個課程包（講師必讀）

## 這個資料夾是什麼
把整套系統切成**一堂一功能**的「閘門檔」。每個 `Lx.md` 是一堂課的控制腳本：框死範圍、嵌入設定關卡、結尾硬性 STOP。學員用 Antigravity（`agy`）一課一課長出系統，不會一口氣跑完。

## 開課三步驟（每堂都一樣）
1. **開場**（貼給 agy）：
   ```
   讀 CLAUDE.md 第 0 節與 安裝包/。本次只執行 lessons/Lx.md 這一課，
   完成它的「✅ 驗收」後停止並回報，不要碰下一課或之後任何功能。先回我「OK」。
   ```
2. **走課文**：照 `Lx.md`。遇到 🧰（設定關卡）就停下，老師去 Console 做、貼金鑰，再對 agy 說「完成，繼續」。
3. **驗收 → commit**：過了 ✅ 驗收才 commit（commit/push 由本機 agy/老師做）。然後**結束本課**，下次再開下一課。

## 一小時一堂的分課（可依時數合併）
| 課 | 主題 | 設定關卡 🧰 | 時長 |
|---|---|---|---|
| L0 | 網站上線（PWA 空殼） | 建 GitHub repo | ~60m |
| L1 | 接雲端 Firestore | Firebase 專案/正式版/開登入/拿 config/貼規則 | ~60m |
| L2 | 真實課表上線 | （無，課表資料） | ~50m |
| L3 | LIFF 認人＋查堂數 | LINE Login＋LIFF（同 Provider）＋圖文選單 | ~60m |
| L4 | 預約引擎（核心） | （無） | ~60m |
| L5 | 取消預約 | （無） | ~40m |
| L6 | 老師後台＋購課到期 | （無） | ~60m |
| L7A | 停課定向通知 | Cloudflare Worker＋Messaging token＋Secret | ~60m |
| L7B | 自動到期提醒（Cron） | service account 金鑰＋Cron | ~60m |
| L8 | 收尾＋交接 | （無） | ~40m |

- **2.5 小時場**：可一次跑 2–3 堂（如 L0+L1+L2，或 L4+L5）。
- **1 小時場**：一次一堂；L7B 最硬，務必獨立一堂。

## 鐵律（為什麼這樣設計）
- agentic 代理會把看得到的目標一次做完 → 所以**靠閘門檔 + STOP**限制，不是靠把字寫多。
- 每課 ship 一個可驗收功能 → 弱模型出錯當場抓到，不會疊成一坨難 debug。
- 詳細點擊步驟在根目錄 `L*-操作手冊.md`；這裡的 `Lx.md` 是「上課控制腳本」，需要細節時叫 agy 去讀對應手冊。
