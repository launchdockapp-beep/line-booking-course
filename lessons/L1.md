# L1 — 接雲端 Firestore　⏱ ~60 分

**本課完課你會看到**：按「雲端連線測試」→ 寫一筆到 Firestore 再讀回；換另一台裝置也讀得到。
詳細見根目錄 `L1-接Firestore-操作手冊.md`。

---

### 🧰 設定關卡 1（老師，課程開始）— 建 Firebase
1. console.firebase.google.com（教室專用 Google 帳號）→ 建專案。
2. Firestore Database → 建立 → 位置 `asia-east1` → **務必選「正式版模式」**。
3. Authentication → Sign-in method → 啟用 **匿名** 與 **Google**。
4. 專案設定 → 新增 Web 應用 → 複製 `firebaseConfig`（只取值）。

### 👩‍💻 給 agy 的指令（這一課，前半）
```
只做 L1：建 firebase-init.js — 用我稍後貼的 firebaseConfig 呼叫 initializeApp、signInAnonymously、建 db（config 先留明顯常數區）。
index.html 載入 compat SDK（app/firestore/auth 10.x）+ 一顆「雲端連線測試」按鈕（寫 config/ping={at:serverTimestamp} 再讀回顯示）。
sw.js 不要快取 Firebase 外部請求。做完停下等我貼 config 與設定授權網域。不要做 L2 課表。
```

### 🧰 設定關卡 2（老師，課中）— 貼 config＋授權網域＋規則
1. 把 `firebaseConfig` 值貼進 firebase-init.js 的常數區（叫 agy 幫貼也行）。
2. Authentication → Settings → 授權網域加 `你的帳號.github.io` 與 `localhost`。
3. Firestore → 規則 → 貼 `安裝包/firestore.rules`，把 `STUDIO_EMAIL@gmail.com` 換成教室信箱 → **發布**。

### ✅ 驗收
- 按測試按鈕顯示「連線成功＋時間」；Firebase Console 看到 `config/ping`。
- 換另一裝置開同網址也讀得到 → 證明在雲端。

本課結束。等「開始 L2」。
