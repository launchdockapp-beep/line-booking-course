### 🧰 設定關卡 1（老師，課程開始）— 建 LINE Login channel ＋ LIFF
> 🔑 鐵律：LINE Login channel 必須建在**你 OA（官方帳號）所在的同一個 Provider** 底下，會員 userId 兩邊才一致、L7 推播才推得到人。**絕不要新開一個 Provider。**

**① 建 LINE Login channel**
1. 開 https://developers.line.biz/console/ → 用管理 OA 的帳號登入。
2. 左側點進**你 OA 所屬的那個 Provider**（進去能看到既有的 Messaging API channel＝你的 OA，就是對的 Provider）。
3. 點 **「Create a new channel」** → 類型選 **「LINE Login」** → 填名稱/圖示/地區 → 同意條款 → 建立。

**② 在該 channel 加 LIFF app**
1. 進入剛建的 LINE Login channel → 點上方 **「LIFF」** 分頁 → **「Add」**。
2. 填：**Endpoint URL** = 你的 GitHub Pages 網址（例 `https://<你>.github.io/line-booking-course/`）；**Size** = `Full`；**Scopes** 勾 `profile`、`openid`。
3. 按「Add」→ 畫面出現 **LIFF ID**（形如 `2007xxxxxx-xxxxxxxx`）→ 記下來（設定關卡 2 要填）。

> LINE 這段沒有好用的 CLI（LIFF CLI 反而多裝多坑），就用 Console。重點是別建錯 Provider。
