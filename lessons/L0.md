### 🧰 設定關卡 1（老師，課程開始）— 建 GitHub repo ＋ gh 授權
agy 做不到，要你登入 GitHub：
1. github.com → New repository → 名稱自取 → **Public**（免費帳號要跑 Pages）→ 不勾任何初始檔 → Create。
2. 一次性 `gh` 授權（**取代所有 PAT／token 做法，零明碼**）：
   ```
   brew install gh        # 若尚未安裝
   gh auth login          # 選 GitHub.com → HTTPS → Login with a web browser
   ```
3. 終端機接遠端：
   ```
   git init && git branch -M main
   git remote add origin https://github.com/<你>/<repo>.git
   ```
> 先別 push，等本課做完再 push。**永遠不要產 PAT、不要建 .env、不要寫 deploy.sh**——push 靠 gh 憑證即可。
> 🔧 **多帳號坑**：若這台電腦的 gh 登過多個 GitHub 帳號，push 可能 403（用到別的帳號）。先 `gh auth status` 看 active 是誰，不對就 `gh auth switch --user <你的帳號>`。
