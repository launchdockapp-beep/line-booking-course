# LINE 課程預約系統 — 課程包（學員乾淨起點）

這是**開課用的乾淨 repo**：系統還沒建，學員用 Antigravity（`agy`）依 `lessons/` 一課一課建起。

## 內容
- `CLAUDE.md`：專案記憶＋**教學模式鐵律**（agy 啟動自動讀，先讀第 0 節）。
- `lessons/`：每堂一個閘門檔（L0…L8），框死範圍＋設定關卡＋驗收＋STOP。先讀 `lessons/00-如何使用這個課程包.md`。
- `安裝包/`：對照真理（資料模型、安全規則、扣堂交易參考碼）。學員不從零寫核心，只接上。

## 開課方式
1. 學員 clone 這個 repo。
2. `cd` 進去 → `agy`。
3. 每堂貼開課咒語（見 `CLAUDE.md` 第 0 節），只跑指定的 `lessons/Lx.md`，做完停。

> 完整可運作的「老師解答版」在另一個 repo（`line-booking-system`），出狀況可對照。
